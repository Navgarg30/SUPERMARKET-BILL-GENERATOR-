# SUPERMARKET-BILL-GENERATOR-
- An interface for generating bill in a supermarket
  
## Features
- Interactive UI
- Seprate bill area
- Unique bill no.
- Bill saved in Database
- Product Details
- Predefined price of product
- Easy UI
- Last sakes graph
- Bill history
- Viewing My Prev Bills
- Viewing Order Details & Status
